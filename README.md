# Kishor Kumar Mahato - Portfolio

Personal portfolio website showcasing my work as a Senior Full-Stack Developer.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Run tests
npm test
```
