# Prezlock: Custom Shopify Theme for a Smart Padlock Brand

![Prezlock Store](/projects/images/prezlock-store.jpg)

## The Project

Prezlock sells smart fingerprint padlocks. They needed a Shopify store that looked premium and worked globally. I modified the Dawn theme heavily: interactive product pages, multi-currency support for 5 currencies, custom slideshow sections, and custom Liquid throughout.

**Live:** [prezlock.com](https://www.prezlock.com/)

---

## What I Did

- Customized Dawn theme to match the client's exact design specs
- Built interactive product pages showcasing the fingerprint padlock features
- Added multi-currency support (5 currencies) for international customers
- Created custom slideshow sections for the homepage
- Wrote custom Liquid code for product displays and checkout flow

![Prezlock Product Page](/projects/images/prezlock-product.jpg)

---

## The Hard Parts

**Multi-currency that actually works.** Supporting 5 currencies means prices, checkout, and payment all need to adapt. This required careful Liquid logic and testing across all currency combinations.

**Showcasing a tech product on Shopify.** A smart padlock isn't a t-shirt. The product pages needed to communicate fingerprint technology, battery life, and security features in a way that standard Shopify templates don't support. Custom Liquid sections handled this.

---

## Tech Stack

Shopify (Dawn theme), Liquid, HTML/CSS, JavaScript
