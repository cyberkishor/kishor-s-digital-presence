# Yeti Pet: Custom Shopify Theme for Premium Pet Products

![Yeti Pet Store](images/yeti.avif)

## The Project

Yeti Pet sells premium pet products. The client provided detailed design mockups and I turned them into a fully working Shopify theme. The focus was on getting the designs right down to the details, making it responsive, and building it so the store owner can manage content independently.

**Live:** [yeti.pet](https://yeti.pet/)

---

## What I Did

- Built a custom Shopify theme from the client's design mockups
- Matched the designs accurately across all page types (homepage, collections, products, cart)
- Created modular admin sections so the client can rearrange and update content
- Optimized assets and code for fast page loads
- Tested across devices and browsers


---

## The Hard Parts

**Translating complex designs into Liquid.** Some design elements don't have a straightforward Shopify equivalent. Built custom Liquid components to achieve the right look while keeping the code maintainable.

**Asset optimization.** Pet product images tend to be large. Set up responsive image handling and lazy loading to keep the store fast without compromising visual quality.

Shopify, Liquid, HTML/CSS, JavaScript
