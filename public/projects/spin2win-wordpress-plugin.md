# Spin2Win: WooCommerce Wheel of Fortune Plugin

![Spin2Win Plugin](images/spin2win.png)

## What It Does

A WordPress plugin that adds a spin-the-wheel popup to WooCommerce stores. Visitors spin for a chance to win discounts, free shipping, or gift cards. Store owners configure the prizes, set the probability for each one, and customize the look. It also captures email addresses through MailChimp integration, turning the game into a lead generation tool.

---

## Features

- **Prize configuration:** Set what each wheel segment offers (5% off, 10% off, free shipping, try again, etc.)
- **Custom probabilities:** Control exactly how often each prize hits
- **CSS customization:** Match the wheel's look to your store's branding
- **Live admin preview:** See changes in real-time on the backend before publishing
- **MailChimp integration:** Collect email addresses before the spin
