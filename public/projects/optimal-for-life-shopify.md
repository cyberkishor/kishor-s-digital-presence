# Optimal For Life: Shopify Store Customization

![Optimal For Life Store](images/optimal.avif)

## The Project

Optimal For Life is a health and wellness e-commerce brand. They had a default Shopify theme that didn't match their brand at all. I customized it into something that actually represents who they are, with a clean UI, proper responsive behavior, and good page speed.

**Live:** [optimalforlife.com](https://optimalforlife.com/)

---

## What I Did

- Reworked the default theme to match the client's brand identity
- Built out homepage, collection pages, product pages, and content pages
- Made everything responsive across desktop, tablet, and mobile
- Set up modular sections the client can edit themselves in Shopify admin
- Optimized images and cleaned up JavaScript to improve load times



---

## The Hard Parts

**Working within Shopify's constraints.** The client's designs didn't always map cleanly to what Shopify's theme system allows. Required creative use of Liquid and CSS to get the look right without hacking things together.

**Making it maintainable.** The client needed to update content regularly without developer help. Everything is built with Shopify's section editor in mind so they can make changes directly.

Shopify, Liquid, JavaScript, CSS
